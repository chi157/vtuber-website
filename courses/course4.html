📚 第四課：流程控制

🎯 本課目標

學習如何控制程式的執行流程

掌握條件判斷與迴圈的使用方法

🔀 if 條件判斷

根據條件決定要執行哪段程式碼。

1️ 基本 if 語句
age = 18 if age >= 18: print("你是成年人")
⚠️ 注意：條件後要加冒號 :
⚠️ if 內的程式碼必須縮排
2 if-else 語句
if age >= 18: print("你是成年人") else: print("你是未成年人")
3 if-elif-else 語句
score = 85 if score >= 90: print("優秀") elif score >= 80: print("良好") elif score >= 60: print("及格") else: print("不及格")
💡 elif = else if 的縮寫
💡 可以有多個 elif，但只能有一個 else

🔁 巢狀條件

在 if 語句中包含另一個 if 語句。

1️ 範例
age = 20 has_id = True if age >= 18: if has_id: print("可以進入") else: print("需要出示證件") else: print("未成年，不可進入")

🔄 for 迴圈

重複執行特定次數的程式碼。

1️ 基本 for 迴圈
for i in range(5): print(i) # 輸出 0, 1, 2, 3, 4
💡 range(5) 產生 0 到 4 的數字
2 range() 函數用法
range(5) # 0, 1, 2, 3, 4
range(1, 6) # 1, 2, 3, 4, 5（起始, 結束）
range(0, 10, 2) # 0, 2, 4, 6, 8（起始, 結束, 步長）
range(10, 0, -1) # 10, 9, 8, ..., 1（倒數）
⚠️ 結束值不包含在內
3 遍歷字串
for char in "Hello": print(char) # 輸出 H e l l o（每個字母）
4 累加計算
total = 0 for i in range(1, 11): total += i # 累加 1 到 10 print(total) # 55

🔁 while 迴圈

當條件為真時，持續執行程式碼。

1️ 基本 while 迴圈
count = 0 while count < 5: print(count) count += 1 # 重要！要改變條件
⚠️ 必須確保條件會變成 False，否則無限迴圈
2 使用者輸入驗證
password = "" while password != "1234": password = input("請輸入密碼: ") print("密碼正確！")
3 無限迴圈（慎用）
while True: user_input = input("輸入 'q' 離開: ") if user_input == 'q': break # 跳出迴圈
💡 搭配 break 使用才能跳出

⏸️ break 和 continue

控制迴圈的執行流程。

1️ break - 跳出迴圈
立即結束整個迴圈
for i in range(10): if i == 5: break # 當 i=5 時跳出 print(i) # 輸出 0, 1, 2, 3, 4
2 continue - 跳過本次迴圈
跳過本次，繼續下一次迴圈
for i in range(10): if i % 2 == 0: continue # 跳過偶數 print(i) # 輸出 1, 3, 5, 7, 9（只有奇數）
3 實用範例：尋找質數
num = 17 is_prime = True for i in range(2, num): if num % i == 0: is_prime = False break # 找到因數就跳出

🔄 巢狀迴圈

在迴圈中包含另一個迴圈。

1️ 九九乘法表
for i in range(1, 10): for j in range(1, 10): print(f"{i} x {j} = {i*j}")
2 數字矩陣
for i in range(1, 4): for j in range(1, 4): print(i * j, end=" ") print() # 換行
輸出：
1 2 3
2 4 6
3 6 9

📊 for vs while 選擇

使用 for 的時機
• 知道確切的迴圈次數
• 遍歷序列（字串、列表等）
• 計數器型迴圈
使用 while 的時機
• 不知道迴圈次數
• 根據條件決定是否繼續
• 使用者輸入驗證
• 需要無限迴圈（配合 break）

✍️ 實作練習

練習一：成績等級判斷
輸入一個分數（0-100）
判斷等級：A (90+), B (80+), C (70+), D (60+), F (<60)
練習二：猜數字遊戲
設定一個 1-100 的答案
讓使用者猜測，提示太大或太小
猜對後顯示猜了幾次
練習三：階乘計算
輸入一個正整數 n
計算 n! = 1 × 2 × 3 × ... × n
例如：5! = 120
練習四：購物清單統計
讓使用者輸入多個商品價格（輸入 0 結束）
計算總金額、商品數量、平均價格
提示：使用 while 迴圈和累加變數
練習五：倒數計時器
輸入倒數秒數（例如 10）
從該數字倒數到 0，每秒顯示剩餘時間
到達 0 時顯示「時間到！」
提示
注意縮排，每層都要正確
使用 break 可以提高效率
巢狀迴圈要小心無限迴圈
