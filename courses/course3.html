📚 第三課：資料型態與運算

🎯 本課目標

深入了解 Python 的資料型態

學習各種運算符號的使用方法

🔢 數值型態詳解

1️ 整數 (int)
正整數：num = 100
負整數：num = -50
零：num = 0
大數字可用底線分隔：num = 1_000_000
2️ 浮點數 (float)
小數：price = 19.99
科學記號：num = 1.5e3 # 1500.0
⚠️ 注意：浮點數可能有精度問題
0.1 + 0.2 結果是 0.30000000000000004

➕ 算術運算子

➕ 加法
10 + 5 # 結果：15
➖ 減法
10 - 5 # 結果：5
✖️ 乘法
10 * 5 # 結果：50
➗ 除法（浮點數）
10 / 3 # 結果：3.3333...
⚠️ 結果永遠是浮點數
➗ 整數除法（取商）
10 // 3 # 結果：3
只保留整數部分，捨去小數
📐 取餘數（模運算）
10 % 3 # 結果：1
常用於判斷奇偶數：num % 2 == 0 為偶數
⚡ 次方（指數）
2 ** 3 # 2的3次方 = 8
10 ** 0.5 # 開根號 ≈ 3.162

🔄 複合指定運算子

簡化「先運算再指定」的寫法。

📝 基本用法
x = 10
x += 5 # 等同於 x = x + 5，結果：15
x -= 3 # 等同於 x = x - 3，結果：12
x *= 2 # 等同於 x = x * 2，結果：24
x /= 4 # 等同於 x = x / 4，結果：6.0
x //= 2 # 等同於 x = x // 2，結果：3
x %= 2 # 等同於 x = x % 2，結果：1
x **= 3 # 等同於 x = x ** 3，結果：1

⚖️ 比較運算子

用於比較兩個值，結果為 True 或 False。

🔍 六種比較運算子
5 == 5 # 等於，結果：True
5 != 3 # 不等於，結果：True
5 > 3 # 大於，結果：True
5 < 3 # 小於，結果：False
5 >= 5 # 大於等於，結果：True
5 <= 3 # 小於等於，結果：False
⚠️ 常見錯誤
等於是 == 不是 =
= 是指定，== 是比較

🔗 邏輯運算子

用於組合多個條件判斷。

🔵 and（且）
兩邊都是 True 才是 True
True and True # True
True and False # False
age >= 18 and age <= 65
🟢 or（或）
任一邊是 True 就是 True
True or False # True
False or False # False
age < 18 or age > 65
🔴 not（非）
反轉布林值
not True # False
not False # True
not (age >= 18)

📝 字串運算

➕ 字串串接（+）
"Hello" + " " + "World" # "Hello World"
"Python" + "3" # "Python3"
✖️ 字串重複（*）
"Ha" * 3 # "HaHaHa"
"-" * 20 # "--------------------"
📏 字串長度 len()
len("Hello") # 5
len("柒柒") # 2（中文也算一個字元）
🔍 檢查子字串 in
"ello" in "Hello" # True
"xyz" in "Hello" # False

🔄 型態轉換

將一種資料型態轉換成另一種。

🔢 轉整數 int()
int("123") # 123
int(3.14) # 3（捨去小數）
int(True) # 1
⚠️ int("3.14") 會錯誤！需先轉 float
🔢 轉浮點數 float()
float("3.14") # 3.14
float(100) # 100.0
float(True) # 1.0
📝 轉字串 str()
str(123) # "123"
str(3.14) # "3.14"
str(True) # "True"
✔️ 轉布林值 bool()
非零數字 → True：bool(100) # True
零 → False：bool(0) # False
非空字串 → True：bool("Hi") # True
空字串 → False：bool("") # False

⚡ 運算優先順序

和數學相同，有優先順序規則。

1️ 括號 ( ) 最優先
2️ 指數 **
3️ 乘除、取餘 * / // %
4️ 加減 + -
5️ 比較運算 == != > < >= <=
6️ 邏輯運算 not and or
📝 範例
2 + 3 * 4 # 14（先乘後加）
(2 + 3) * 4 # 20（括號優先）
5 > 3 and 10 < 20 # True

✍️ 實作練習

練習一：溫度轉換
輸入攝氏溫度，轉換為華氏溫度
公式：華氏 = 攝氏 × 9/5 + 32
練習二：BMI 計算機
輸入身高（公分）和體重（公斤）
計算 BMI = 體重 / (身高/100)²
練習三：奇偶數判斷
輸入一個整數
使用 % 運算判斷是奇數還是偶數
💡 提示
注意括號的使用
記得做型態轉換
使用 f-string 美化輸出

掌握這些運算後，你就能進行各種計算和判斷了！
