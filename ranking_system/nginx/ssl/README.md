# SSL certificates will be placed here
# Use Let's Encrypt or Cloudflare Origin certificates

# For Let's Encrypt:
# 1. Install certbot
# 2. Run: certbot certonly --webroot -w /var/www/certbot -d your-domain.com
# 3. Copy fullchain.pem and privkey.pem here

# For Cloudflare Origin certificates:
# 1. Go to Cloudflare Dashboard > SSL/TLS > Origin Server
# 2. Create Certificate
# 3. Save as fullchain.pem and privkey.pem
