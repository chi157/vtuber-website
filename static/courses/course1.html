📚 第一課：Python 環境安裝與設定

🐍 什麼是 Python？

Python 是一種簡單易學、功能強大的程式語言，適合初學者入門。

特色：

• 語法簡潔，容易閱讀和理解
• 應用廣泛：網站開發、資料分析、AI、自動化等
• 擁有龐大的社群和豐富的套件庫

🎯 什麼是 Anaconda？

Anaconda 是一個 Python 發行版本，專為資料科學和機器學習設計。

包含內容：

• Python 核心程式
• 超過 1500+ 個常用套件（NumPy、Pandas 等）
• Spyder、Jupyter Notebook 等開發工具
• Conda 套件管理器

💡 為什麼使用 Anaconda？

✅ 一鍵安裝：不需要分別安裝 Python 和各種套件，省時省力
📦 套件管理：Conda 可以輕鬆管理和更新套件，避免版本衝突
🔧 環境隔離：可以為不同專案建立獨立的 Python 環境
🎓 適合學習：內建教學所需的所有工具，初學者友善

💻 Windows 安裝步驟

1 下載 Anaconda
前往官網：https://www.anaconda.com/download
選擇 Windows 版本，下載 64-Bit Installer
2 執行安裝程式
雙擊下載的 .exe 檔案
選擇「Just Me」（僅為目前使用者安裝）
安裝路徑建議使用預設值
⚠️ 重要：勾選「Add Anaconda to my PATH environment variable」
3️⃣ 驗證安裝
開啟「Anaconda Prompt」（從開始功能表搜尋）
輸入指令：python --version
應該會顯示 Python 版本號（如：Python 3.11.x）

🍎 macOS 安裝步驟

1️ 下載 Anaconda
前往官網：https://www.anaconda.com/download
選擇 macOS 版本（Intel 或 Apple Silicon M1/M2）
2️ 執行安裝程式
雙擊下載的 .pkg 檔案
依照安裝精靈指示完成安裝
安裝路徑通常為：/Users/你的使用者名稱/anaconda3
3️ 驗證安裝
開啟「終端機」（Terminal）
輸入指令：python --version
應該會顯示 Python 版本號

🔧 建立與管理 Conda 環境

環境隔離可以讓不同專案使用不同版本的 Python 和套件，避免衝突。

📝 建立新環境
開啟 Anaconda Prompt (Windows) 或 Terminal (Mac)
輸入：conda create -n myenv python=3.11
（myenv 是環境名稱，可自訂；3.11 是 Python 版本）
▶️ 啟動環境
Windows / Mac：conda activate myenv
📋 查看所有環境
conda env list
⏹️ 退出環境
conda deactivate

🖥️ Spyder IDE 介紹

Spyder 是 Anaconda 內建的 Python 開發環境，特別適合科學計算和資料分析。

⭐ 主要特色：
• 介面類似 MATLAB、RStudio，易於上手
• 內建變數檢視器，可即時查看資料
• 整合 IPython 控制台，支援互動式程式開發
• 程式碼自動完成和除錯功能

🚀 在指定環境中啟動 Spyder

⚠️ 重要：必須在正確的 Conda 環境中啟動 Spyder，才能使用該環境的套件。

1️ Windows 啟動方式
開啟「Anaconda Prompt」
啟動環境：conda activate myenv
啟動 Spyder：spyder
2️ macOS 啟動方式
開啟「終端機」（Terminal）
啟動環境：conda activate myenv
啟動 Spyder：spyder
✅ 確認環境正確
Spyder 啟動後，在 IPython 控制台輸入：
import sys; print(sys.executable)
應顯示你的環境路徑，確認是在正確的環境中

💡 常見問題

❓ Q：直接從開始功能表點選 Spyder 可以嗎？
A：可以，但會使用 base 環境。建議為每個專案建立獨立環境。
❓ Q：如何在 Spyder 中切換環境？
A：必須關閉 Spyder，在命令列切換環境後重新啟動。
❓ Q：找不到 spyder 指令怎麼辦？
A：在該環境中安裝 Spyder：conda install spyder

完成安裝後，就可以開始學習 Python 了！
