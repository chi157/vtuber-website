📚 第二課：Python 基礎語法

🎯 本課目標

學習 Python 程式設計的基本語法規則

了解如何撰寫你的第一個 Python 程式

💬 註解 (Comments)

註解是給人看的說明文字，程式執行時會被忽略。

1️ 單行註解
使用 # 符號
範例：# 這是註解
print("Hello") # 這也是註解
2️⃣ 多行註解
使用三個引號 """ 或 '''
範例：
"""
這是多行註解
可以寫很多行
"""

📝 變數 (Variables)

變數就像一個容器，用來儲存資料。

✏️ 變數命名規則
• 只能包含字母、數字、底線 (_)
• 不能以數字開頭
• 區分大小寫 (name 和 Name 是不同變數)
• 不能使用 Python 保留字 (if, for, while 等)
📌 變數賦值
name = "柒柒" # 字串
age = 20 # 整數
height = 165.5 # 浮點數
is_student = True # 布林值
🔄 多重賦值
x = y = z = 0 # 同時賦予相同值
a, b, c = 1, 2, 3 # 同時賦予不同值

🖨️ 輸出 print()

使用 print() 函數將資料顯示在螢幕上。

1️⃣ 基本輸出
print("Hello, World!")
print(123)
2️⃣ 輸出變數
name = "柒柒"
print(name) # 輸出：柒柒
3️⃣ 輸出多個值
print("我是", name) # 用逗號分隔
print("年齡:", age, "歲")
4️⃣ 格式化輸出
print(f"我是{name}，今年{age}歲") # f-string
print("我是{}".format(name)) # format()

⌨️ 輸入 input()

使用 input() 函數讓使用者輸入資料。

📥 基本輸入
name = input("請輸入你的名字: ")
print("你好,", name)
⚠️ 重要：input() 回傳的是字串
如果要輸入數字，需要轉換型態
age = int(input("請輸入年齡: ")) # 轉整數
height = float(input("請輸入身高: ")) # 轉浮點數

📏 縮排 (Indentation)

⚠️ Python 使用縮排來定義程式碼區塊，這是最重要的語法規則！

✅ 正確的縮排
通常使用 4 個空格或 1 個 Tab
同一層級的程式碼必須有相同的縮排
Spyder 預設按 Tab 鍵會自動轉為 4 個空格
❌ 常見錯誤
混用空格和 Tab（會導致錯誤）
縮排不一致（會導致 IndentationError）

🔤 基本資料型態

📝 字串 (str)
用單引號或雙引號包起來的文字
text = "Hello" 或 text = 'Hello'
🔢 整數 (int)
沒有小數點的數字
num = 100
🔢 浮點數 (float)
有小數點的數字
price = 99.99
✔️ 布林值 (bool)
只有兩個值：True 或 False
⚠️ 注意：首字母必須大寫

🔍 查看資料型態

🛠️ 使用 type() 函數
print(type("Hello")) #
print(type(123)) #
print(type(3.14)) #
print(type(True)) #
